# 🚕 Book A Taxi

Taxi booking application frontend built with the MERN stack.

## Features

- Ride-share-style UI using Tailwind
- Google Maps pickup and dropoff selection
- Ride price estimator
- Booking form validation
- REST API using json-server

## Tech Stack

- React
- TailwindCSS
- React Router
- Axios
- Google Maps API
- json-server

## Running the Project

### Frontend
*Note: Front-end requires a Google API key to function. [Get a demo API key.](https://mapsplatform.google.com/maps-demo-key/). Once you have the key, create a .env.local file in the frontend directory and add the key as:*

`VITE_GOOGLE_MAPS_API_KEY=YourKeyGoesHere`

To run frontend project:
```
npm install
npm run dev
```

### Backend

```
npm start
```
