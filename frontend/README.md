# Book_A_Taxi

Taxi booking application frontend built with the MERN stack.

## Features

- Uber-style UI using Tailwind
- Google Maps pickup and dropoff selection
- Ride price estimator
- Booking form validation
- REST API using json-server

## Tech Stack

React
TailwindCSS
React Router
Axios
Google Maps API
json-server

## Running the Project

Frontend

npm install
npm run dev

Backend

json-server --watch backend/db.json --port 3001
